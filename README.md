# NODE-AppClima

Console application made with node js receives the name of a city and returns its current weather.

API used to return weather information:

      Openweather: https://openweathermap.org/api

used packages:

    colors: ^1.4.0
    
    inquirer: ^8.2.4
    
    axios: ^0.27.2
    
    dotenv: ^16.0.1
